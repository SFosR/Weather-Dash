# Weather Dashboard
  
# Description
This app allows clients to see the weather outlook for multiple cities for when they want to make travel plans. Clients can search by city and will see the current conditions, 5-day forecast, temperature, humidity levels, wind speed and UV index. It will also save the clients search history for quick access when clients visit the app again. 
  
# Technologies Used
  * JavaScript
  * jQuery
  HTML
  CSS
  OpenWeather API
